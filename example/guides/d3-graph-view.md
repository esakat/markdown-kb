# グラフビューの使い方

ドキュメント間の関連をインタラクティブなグラフで視覚化できます。

## アクセス方法

ヘッダーの **Graph** リンクをクリックするか、`/graph` に直接アクセスします。

## ノードとエッジ

### ノード（丸）

- 各ノードは 1 つのドキュメントを表す
- ノードの大きさはタグの数に比例
- 色はプライマリタグに基づく
- クリックでドキュメント詳細に遷移

### エッジ（線）

| 種類 | 見た目 | 意味 |
|------|--------|------|
| リンク | 青い破線 | `[[wiki-link]]` や `[text](path.md)` |
| 共有タグ | グレーの実線 | 同じタグを持つドキュメント同士 |

## インタラクション

- **ズーム**: マウスホイール / ピンチ
- **パン**: ドラッグ（空白部分）
- **ノード移動**: ドラッグ（ノード上）
- **詳細表示**: ノードクリック
- **タグフィルタ**: 右サイドバーのタグをクリック

## リンクの記法

グラフにエッジを生成するリンクの書き方:

```markdown
<!-- Wiki リンク（.md は自動付与） -->
[[guides/getting-started]]

<!-- Markdown リンク -->
[検索のコツ](guides/search-tips.md)
```

**外部 URL** (`https://...`) やアンカー (`#section`) はグラフに含まれません。

## タグクラウド

サイドバーのタグクラウドでは:

- タグのフォントサイズが出現頻度に比例
- クリックでそのタグを持つノードだけにフィルタ
- もう一度クリックで解除
